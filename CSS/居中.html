        # 居中

        # 水平居中

        # 1. 行内元素

        text-align 只对文本和行内元素生效

        • 父元素设置text-align:center
        <div class="parent">
  <span>text-align:center</span>
</div>

        .parent {
  text-align: center;
}
        text-align:center

        # 2. 块级元素

        子元素设置margin:0 auto;

        .child {
  width: 100px;
  margin: 0 auto;
}
        margin:0 auto

        # 3. 通杀 flex

        .parent1 {
  display: flex;
  justify-content: center;
}
        justify-content:center

        # 垂直居中

        # 1. 行内元素

        设置父元素line-height和高度一样

        原理：line-height=文字内容高度+上间距+下间距，文字本身在自己的行框内居中。如果 line-height=高度，那么它上下间距一样，位置没有任何变化。

        width: 50px;
line-height: 50px;
        line-height

        # 2. 块级元素

        父级设置position:relative，子元素使用绝对定位+transform 可以在宽高未知的情况下实现居中

        .parentV {
  position: relative;
}
.child-abs {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}
        absolute

        # 3. 通杀 flex

        .alignItems {
  display: flex;
  align-items: center;
}
        align-items:center

        # 垂直水平居中

        # 1. 行内元素

        结合前面两种写法,设置text-align:center;,line-height为父元素高度

        line-height

        # 2. 块级元素

        将绝对定位法补充完整，加上水平方向

        .child-abs {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
        absolute

        # 3. 通杀 flex

        父元素设置display:flex;justify-content:center;align-items:center;

        flex

        # 总结

        flex 是首选方案，IE9 以上全支持，点击查看 flex 兼容性
